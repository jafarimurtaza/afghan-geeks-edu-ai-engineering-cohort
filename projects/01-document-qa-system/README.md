# Capstone 1 — Document Q&A System

**Session 14**

A document question-answering system powered by a local LLM via Ollama. Load documents, ask questions, get grounded answers.

## Goals

- Ingest and chunk documents.
- Retrieve relevant context for a question.
- Generate answers with a local model through Ollama.

## Getting started

```bash
ollama pull llama3.2
ollama serve
```
